Hola soy Ariel Fajardo

Desarrollo sitios y aplicaciones web

Conoce mas sobre mí

¡Hola! Soy Ariel Fajardo, un apasionado desarrollador web con un enfoque creativo y una pasión por la creación de experiencias digitales impactantes. Mi enfoque principal es no solo crear sitios web funcionales y visualmente atractivos, sino también garantizar una experiencia de usuario excepcional. Me comprometo a utilizar las últimas tecnologías y mejores prácticas de desarrollo para construir plataformas web intuitivas que no solo cumplan, sino que superen las expectativas de los usuarios.

Desarrollo web Front-End

Como desarrollador front-end, creo interfaces web atractivas y funcionales. Me especializo en HTML, CSS y JavaScript para ofrecer experiencias de usuario dinámicas y de alta calidad.

HTML JavaScript Bootstrap CSS

Aplicaciones Web

Mi trabajo en desarrollo de aplicaciones web se enfoca en crear soluciones digitales eficientes y atractivas. Utilizo tecnologías modernas para ofrecer experiencias interactivas y funcionales a los usuarios.

Flask Python Sql

Desarrollo web Back-end

En el desarrollo back-end, diseño y construyo servidores robustos y eficientes que potencian la funcionalidad y el rendimiento de las aplicaciones web, manejando la lógica y los datos.

Sqllite3 Python
